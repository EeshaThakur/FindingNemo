# FindingNemo – AI for Finding the Missing, Bringing Hope to Families

FindingNemo is an AI-powered missing person detection system that uses Computer Vision and Face Recognition to identify missing individuals from live camera feeds and uploaded surveillance videos. The system enables faster identification by automating the process of monitoring and analyzing visual data.

## Features

* Missing person registration and profile management
* Multiple image dataset creation
* Face detection using OpenCV Haar Cascade
* Face recognition using LBPH algorithm
* Live camera monitoring
* Surveillance video scanning
* Automatic email notifications on detection
* Detection history and status tracking
* Location logging and investigation updates

## Tech Stack

**Frontend:** React.js, Tailwind CSS
**Backend:** Django, Django REST Framework
**Database:** SQLite
**AI & Computer Vision:** OpenCV, Haar Cascade, LBPH Face Recognition

## Workflow

1. Register a missing person and upload multiple photographs.
2. Train the face recognition model using the collected dataset.
3. Monitor live cameras or upload surveillance videos for analysis.
4. Detect matching faces and generate alerts.
5. Update status, store detection logs, and notify the reporter via email.

## Future Enhancements

* Integration with police CCTV networks
* Deep learning-based recognition using FaceNet and DeepFace
* Mobile application support
* Cloud-based real-time monitoring

**FindingNemo aims to leverage Artificial Intelligence and Computer Vision to reduce the time required to locate missing persons and bring hope to families searching for their loved ones.**
